# EasyDarwin开源流媒体服务器

![snapshot](http://ww1.sinaimg.cn/large/79414a05ly1fwzq9dirmij212u0ipwg9.jpg)
 
## 主要功能特点

- 基于Golang开发维护

- 支持windows、linux平台

- 接收RTSP流推送

- RTSP流分发

- 关键帧缓存

- 秒开画面

- Web后台管理

- 分布式负载均衡
